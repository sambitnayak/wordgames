# Word Games

A kid-friendly client-side vocabulary practice game built with HTML, CSS, and JavaScript.

## Features

- Two word games: Mix-up and Blanked out.
- Profile-based score tracking stored in browser local storage.
- Upload a `.txt` word list with one word per line for the current session.
- Words are normalized to uppercase during play.
- Three visual themes: Sunny, Space, and Cute.
- Runs directly in the browser with no server or build step required.

Open `index.html` in a browser to play.
